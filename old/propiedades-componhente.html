# Propiedades componente

Existen dos tipos de propiedades:

  • Simples

    Aquellas que son naturales de java o que no son un objeto en si.

  • Complejas

    Aquellas que son un objeto nuevo creado por alguien y que tiene sus propieas caraterísticas y comportamiento.

# Añadir una propiedad compleja

Ahora crearemos una nueva clase la cual la llamaremos ColorComplejo la cual almacenara dos colores que representaran cada uno el color de texto y el de fondo de nuestro botón.

Esta clase tambien deberá implementar la interfaz Serializable

public class ColorComplejo implements Serializable{
    private Color colorFondo, colorTexto;

    public ColorComplejo(){
        colorFondo = Color.GRAY;
        colorTexto = Color.BLACK;
    }

    public Color getColorFondo() {
        return colorFondo;
    }

    public void setColorFondo(Color colorFondo) {
        this.colorFondo = colorFondo;
    }

    public Color getColorTexto() {
        return colorTexto;
    }

    public void setColorTexto(Color colorTexto) {
        this.colorTexto = colorTexto;
    }
}

Ahora añadiremos esta propiedad a nuestro BotonPersonalizado



 



 




 










 
 
 
 
 
 
 
 


public class BotonPersonalizado extends JButton implements Serializable{
    private int limiteClicks = -1;
    private ColorComplejo colorComplejo;
    
    public BotonPersonalizado(String texto){
        super(texto);
        colorComplejo = new ColorComplejo();
    }
    
    public BotonPersonalizado(){
        super();
        colorComplejo = new ColorComplejo();
    }

    public int getLimiteClicks() {
        return limiteClicks;
    }

    public void setLimiteClicks(int limiteClicks) {
        this.limiteClicks = limiteClicks;
    }

    public ColorComplejo getColorComplejo() {
        return colorComplejo;
    }

    public void setColorComplejo(ColorComplejo colorComplejo) {
        this.colorComplejo = colorComplejo;
    }

}
